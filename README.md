# BLINKIT GROCERY DATA
Developed an interactive Blinkit Sales Analysis Dashboard in Power BI to evaluate grocery retail performance, inventory metrics, and outlet distribution channels. The project centralizes high-volume fast-moving consumer goods (FMCG) transaction data into an enterprise business intelligence platform designed to track sales efficiency and consumer preferences.
